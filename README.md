# odin-calculator
A calculator app for The Odin Project foundations course

## Usage
[[https://carelessantics.github.io/odin-calculator/|LIVE DEMO HERE]]
Works like a regular calculator. Also accepts keyboard input

## Features
The assignment specified the calculator to be able to calculate addition, subtraction, division, and multiplication based on user input, as well as the ability to clear the current function. In addition to these, I also added parentheses, negation, and a backspace/correction feature. The way the logic is structured should allow for custom functions like sin(), pow(), sqrt() to be added relatively easily, but I will leave that for a later date, and they sound a bit like famous last words.
