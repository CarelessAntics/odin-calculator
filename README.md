# odin-calculator
Final project for The Odin Project Foundations course. For this assignment, only the specifications were provided, and this resulting repo is made by me from scratch. 

## Demo
[LIVE DEMO HERE](https://carelessantics.github.io/odin-calculator/)

## Features
The assignment specified the calculator to be able to calculate addition, subtraction, division, and multiplication based on user input, as well as the ability to clear the current function. In addition to these, I also added parentheses, negation, and a backspace/correction feature. The way the logic is structured should allow for custom functions like sin(), pow(), sqrt() to be added relatively easily, but I will leave that for a later date.
